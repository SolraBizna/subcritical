jpeg package

jpeg contains no utilities.

License

The jpeg package is marked as Compatible, which means any game can use it.

Classes

JPEGLoader : GraphicLoader

JPEGLoader loads color or grayscale JIF (*.jpg and *.jpeg) files.

JPEGLoader has no methods it didn't inherit.

jpeg_loader = SubCritical.Construct("JPEGLoader")

JPEGDumper : GraphicDumper

JPEGLoader writes progressive JIF files. The recommended extension for new JIF files is *.jpeg.

jpeg_dumper = SubCritical.Construct("JPEGDumper")
Prepares a JPEGDumper.
dumper:SetQuality(quality)
Sets the quality of produced JPEG data on the standard 0-100 scale.
